auditctl用于配置与审计相关的内核选项

参数 作用
-w 需要监控的文件或是命令
-p 指定监控属性
-k 设置一个关键字用于查询

ausearch查看审计日志

参数 作用
-a 只显示id为指定数字的日志信息
-c 只显示和命令相关的信息
-i 显示详细信息
-k 查看带有关键字的信息

nmap扫描端口

参数 作用
-sT TCP connect()扫描，这种方式会在目标主机的日志中记录大批连接请求和错误信息
-sS 半开扫描，很少有系统能把它记入系统日志。不过，需要Root权限
-sF -sN 秘密FIN数据包扫描、Xmas Tree、Nll描模式
-sP ping扫描，Nmap在扫描端口时，默认都会使用ping扫描，只有主机存活，Nmap才会继续扫描
-sU UDP扫描，但UDP扫描是不可靠的
-sA 这项高级的扫描方法通常用来穿过防火墙的规则集
-sV 探测端口服务版本
-Pn 扫描之前不需要用ping命令，有些防火墙禁止ping命令。可以使用此选项进行扫描
-p 指定端口如"1-65535、 1433、135、 22、80等
-r 使用“-r”选项表示不会随机的选择端口扫描
--iflist 检测主机接口和路由信息
-PA ,-PS 使用TCP ACK和TCP Syn方法来扫描远程主机

truncate 限制文件大小

参数 作用
-s 指定文件大小
