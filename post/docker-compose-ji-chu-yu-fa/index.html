docker-compose基础语法

2020-03-10
参数 用途
version 指定docker-compose版本号
services 定义容器服务名字
image 指定服务的镜像
command 启动后指定的命令
container_name 定义容器名字
depends_on 解决容器依赖、先后启动问题
network 容器网络定义
port 容器映射的端口
volume 挂载卷
labels 定义标签
extra_hosts 往host文件写入主机名
network_mode 容器网络模式
environment 设置环境变量
resources 资源限制
healthcheck 健康检查机制
logging 配置日志服务

还有很多 这里就不一一列举了 有兴趣的朋友可以去github查看
